# Interactive Game Website

This project was created for **Milestone 2 – Interactive Website**. It is a simple website built using **HTML, CSS, and JavaScript** without any external frameworks.

The website includes a landing page and several small JavaScript-based games. The main focus of this project is learning **basic JavaScript interaction**, **DOM manipulation**, and **simple game logic**.

## Games

* **Number Guessing Game** – Guess a random number with limited attempts.
* **Rock Paper Scissors** – Play against the computer with simple scoring.
* **Clicker Game** – Increase score by clicking a button.
